# Java CI/CD

Учебный кроссплатформенный Java-проект c настройкой CI/CD и использованием GitHub Actions.

## Описание
Проект представляет собой небольшое Java-приложение для анализа текста
(подсчёт количества слов, символов, проверка на палиндром) с модульными тестами на базе JUnit 5.
Сборка и тестирование выполняются с помощью Apache Maven.

## CI/CD Пайплайн
В репозитории настроен GitHub Actions pipeline, который включает:

* Автоматическую сборку проекта через Maven и генерацию SBOM (CycloneDX)

* Запуск юнит-тестов (JUnit 5)

* Анализ зависимостей на уязвимости:

    * локальный скан через OWASP Dependency Check

    * анализ SBOM через OWASP Dependency Track

* Кэширование базы NVD для ускорения Dependency Check на GitHub Actions

* Сохранение артефактов анализа (SBOM и отчёты) на 7 дней

Пайплайн автоматически запускается только при pull request и при пуше в соответствующую ветку.
